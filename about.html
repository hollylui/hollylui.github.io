About: Holly

I'm Holly. I come from Hong Kong and I am a mom with a kid. I was Mechanical Engineer in semiconductor field. After struggling for a while, I start studying again. Currently I am studying Web Development in DCI since March 2021.

  • In March, learning Basic Digital Literacy (i.e. Terminal, Markdown, APT, npm, git, Github and SSH), html and UI Basics.
  • In April, learning conventional website layout and navbar.
  • In May, learning wireframe, grid, animation, clip-path, CSS framework: Bootstrap, CSS preprocessors: SASS(SCSS) and Boilerplate & npm.
  • Final project in May: this blog is the final project of UIB and based on the website template: November designed by seaboardthemes.
